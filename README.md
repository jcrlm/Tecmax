# TECMAX

Sistema de servicio técnico y facturación para fotocopiadoras.

## Funciones
- Registro de clientes
- Reporte técnico (Word)
- Factura en PDF
- Numeración automática
- Historial por cliente y fecha
- Control de insumos cambiados
- Evidencia con pruebas de impresión

## Uso
```bash
servicio-tecnico
